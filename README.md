# Description about the Project
Tiles Gallery is a world-class tiling brand in Bangladesh that focuses on blending innovation, quality, and artistry. The website serves as a comprehensive digital catalog and corporate hub for homeowners, architects, and developers looking for high-quality interior and exterior surfacing solutions.  A significant portion of the site is dedicated to their commitment to the environment, people, and community, highlighting efforts to reduce carbon footprints and support their workforce. 


# Purpose
Core Purpose of this website is to redefine living and working spaces by providing sustainable, stylish, and durable tiling solutions that meet international standards.


# npm Packages
First, run the development server:
```bash
npm run dev
npm install lucide-react
npm install next-themes daisyui@latest
npm run build
npm i react-spinners
npm i @contentstack/react-toastify
npm install better-auth
npm install mongodb
npm start
```
